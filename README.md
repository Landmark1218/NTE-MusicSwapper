# NTE Music Swapper
This tool allows you to use your own audio files (WAV, MP3, OGG) as in-game radio music.
https://ayakamods.com/mods/nte-music-swapper.2417/

# Tool Description

This tool allows you to use your own audio files (WAV, MP3, OGG) as in-game radio music.

# Requirements:
Before using the tool, you must install Wwise.

# Download the Wwise Launcher:
Wwise Launcher Download
Install Wwise version 2025.1.7.9143

# How to use:

Click the "Browse" button next to the radio track you want to replace.
Select your desired audio file.
Once everything is ready, click "BUILD MOD".
When a success dialog appears, launch the game.

# Auto-Save & Apply MOD:

If enabled: your MOD will be saved and automatically applied next time.
If disabled: the MOD will reset when you close the tool.

# Important:
Keep this tool running while the game is open.
Closing the tool during gameplay may cause issues.
